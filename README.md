# NCAA D-I Coaching Pipeline & Demographic Analysis (2012–2025)

Interactive dashboard analyzing NCAA Division I coaching pipelines and demographic representation across Women's Basketball, Softball, and Volleyball from 2012 to 2025.

## Project Overview

This project examines the structural pathway from student-athlete to assistant coach to head coach, with a focus on:

- athlete vs. coach demographic representation
- assistant-to-head coach promotion gaps
- female coaching leadership trends
- international athlete coaching pipeline gaps
- long-term diversity change across sports

The analysis is designed as a research-style dashboard for athletic departments, conference offices, and sports governance stakeholders.

## Live Dashboard

[View the dashboard](https://kbsmd-sportsmusicdata.github.io/ncaa-d1-coaching-pipeline-dashboard/)

## Key Questions

- Are NCAA athlete populations diversifying faster than coaching staffs?
- Where does the coaching pipeline break down?
- Which sports show the strongest or weakest promotion equity?
- How large are athlete-to-coach representation gaps by race and gender?

## Main Findings

- Athlete diversity has increased across all three sports since 2012.
- Coaching leadership has not diversified at the same pace.
- Women’s Basketball shows the clearest assistant-to-head coach promotion bottleneck.
- International athletes remain meaningfully underrepresented in coaching roles.
- Staff expansion has widened the assistant pipeline without proportionally improving head coach access.

## Dashboard Features

- Overview tab with system-level trends
- Sport-specific deep dives for WBB, Softball, and Volleyball
- Representation gap analysis
- Promotion pipeline analysis
- Metrics & definitions tab

## Methods

Core methods used in this project include:

- Simpson Diversity Index (SDI)
- coach-to-athlete representation gap analysis
- assistant-to-head coach promotion gap analysis
- longitudinal trend analysis (2012–2025)
- cross-sport benchmarking
